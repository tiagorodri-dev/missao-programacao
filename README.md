## ⚡ Funcionalidades:
<p>A aplicação tem como funcionalidade principal causar uma boa experiência ao usuário, tendo como funcionalidades:</p>
<ul>
  <li>Modo light e dark;</li>
  <li>Responsividade para dispositivos móveis e tablets.</li>

<ul>
